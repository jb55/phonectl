## Usage

```sh
$ export PHONECTLUSER=user@domain.com
$ export PHONECTLPASS=pass
$ export PHONECTLPHONE=phone@domain.com

$ python3 phonectld.py &
$ cp phonectl.sh ~/bin/phonectl

$ phonectl battery
Battery level 84%
```
